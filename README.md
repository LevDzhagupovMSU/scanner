# Media File Scanner
Многопоточный **сканер медиафайлов** для Linux/Unix-систем. Рекурсивно обходит домашнюю директорию ($HOME), определяет тип файлов (аудио, видео, изображения) по расширениям и периодически сохраняет результат в JSON-файл.

---
# Возможности
-- Сканирование всей домашней директории с рекурсивным обходом поддиректорий
-- Многопоточность для ускорения сканирования (по умолчанию 4 потока)
-- Определение и настройка типов файлов по расширению в `type_file.hpp` через `FileExtensionF::file_type`
-- Периодический режим работы с задаваемой частотой
-- Интерактивная остановка по нажатию q или Q
-- Сохранение результатов в файл `~/.media_files` в формате JSON
-- Символические ссылки игнорируются

# Требования
-- Компилятор с поддержкой C++17 
-- CMake
-- nlohmann/json

# Установка:
```bash

git clone https://github.com/LevDzhagupovMSU/scanner.git
cd scanner
```
```bash
mkdir build && cd build
cmake ..
make
```

# Пример запуска
```bash
./media_scanner
```

```bash
Enter Scan Frequency in Seconds
> 10
```

# Структура проекта
-- `header.hpp` – общие заголовки 

-- `type_file.hpp/cpp` – определение типов файлов

-- `scan_result.hpp/cpp` – структура результата сканирования

-- `thread_scan.hpp/cpp` – класс scanner_dir (основная логика)

-- `main.cpp` – точка входа, обработка ввода
